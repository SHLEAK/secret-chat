# WebRTC Peer-to-Peer Chat Prototype

## Project Overview
This is a simple browser-based peer-to-peer chat prototype built using WebRTC.  
The system allows two users to connect directly in their browsers and exchange text messages.  
The server stores who's online.

This project demonstrates the **Realisation** and **Manage & Control** phases of software development:
- Building and testing a simple software system.
- Organizing work using GitHub, branches, issues, and project boards.
- Documenting workflow and reflecting on development.

---

## Features
- Peer-to-peer messaging using WebRTC
- Local database to store chat logs
- Simple and functional UI
- Tested for basic connection and message reliability
